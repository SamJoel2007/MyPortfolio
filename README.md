# SAM Portfolio

A cyberpunk-inspired single-page portfolio website built with HTML and CSS.

## 🚀 About

This project is a personal portfolio landing page with a futuristic green-on-black theme, custom cursor, and animated visual effects. It is designed to showcase skills, projects, and a developer profile in a clean one-page format.

## 📌 Key Features

- Responsive layout built with HTML and CSS
- Bootstrap 5.3 and Font Awesome loaded via CDN
- Animated hero section with typed text styling
- Custom cursor and subtle matrix-style background overlay
- Skills and projects sections with futuristic UI elements
- Mobile-friendly navigation and section scrolling

## 📁 Project Structure

- `index.html` – main portfolio page
- `README.md` – project documentation

## ▶️ Run Locally

1. Clone or download the repository.
2. Open `index.html` in your browser.

> If you want to preview it using a local server, use any simple static-server tool or the built-in Live Server extension in VS Code.

## 💡 Notes

- No build tools or package manager required.
- All styling is contained directly in `index.html`.
- External assets are loaded from CDN links, so internet access is needed for Bootstrap and Font Awesome.
